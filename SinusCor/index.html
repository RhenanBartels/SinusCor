SinusCor

SinusCor is an advanved tool for Heart Rate Variability analysis distributed under a freeware license, which means you can download and share without any payment. Along with classical indeces, SinusCor also allows you to analyze non stationary HRV signals, like those recorded during an exercise bout.

Instalation

To have access to SinusCor, please send an email to rhenan.bartels@gmail.com and we will send a zip file containing the program and some sample signals. Then, you need to download and save the file in a directory of your choice and decompress it.

The code was developed using Matlab, therefore you need Matlab installed in your machine together with the Signal Processing Toolbox. Open Matlab and than change the working directory to the one you choosed to save SinusCor. Now just type SinusCor in the Command Window and the software will be launched.

Requirements

  • Matlab version >= 7.9.0.529 (R2009b)
  • Signal Processing Toolbox

Platforms

  • Window, MacOS, Linux
